## 开发者评价
> 截至2022年4月，人工智能语音合成已经取得了不小的成果，万千音色、多种情绪、呼吸停顿非常接近真人。

## 简介
语音合成一直以来是语言、语音、深度学习及人工智能等领域的热门研究方向，受到了学术界和工业界广泛的关注。P2G工具箱目前集成了来自微软、阿里云、火山的语音合成引擎。
